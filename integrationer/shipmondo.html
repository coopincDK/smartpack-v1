SmartPack + Shipmondo

Integrer Shipmondo med SmartPack og få adgang til alle Shipmondos fragtleverandører i ét flow. Print labels, sporingsnumre og ordrestatussynk, alt kører automatisk.

Kom i gang →Se priser

Hvad gør integrationen?

🌎

Alle Shipmondo-carriers

Shipmondo giver adgang til GLS, PostNord, DAO, DHL, UPS og mange flere via ét integreret API i SmartPack.

🏠

Print labels fra SmartPack

Vælg carrier per ordre og print label direkte fra pakkelinjen. Sporingsnummer synkroniseres automatisk.

📌

Automatisk ordrestatus

Når pakken er afsendt opdateres ordrestatus i din webshop via Shipmondo.

Sådan forbinder du Shipmondo med SmartPack

1

Aktivér Shipmondo

Gå til Integrationer i SmartPack og vælg Shipmondo. Log ind med din Shipmondo-konto.

2

Konfigurér carriers og regler

Sæt standardcarrier og konfigurér eventuelle automatiske fragtvalgsregler.

3

Test og gå live

Send en testforsendelse gennem Shipmondo og verificer hele flowet.
